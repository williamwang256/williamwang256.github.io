Computer Science

CSC301: Introduction to Software Engineering

Winter 2022

Learned about various aspects of software development, from design patterns and clean code to APIs, databases and CI/CD. This course involved working on a fast-moving project with a team, using agile software development principles. We were given a vision for a product and had to go through the process of planning and implementing it all the various parts, including choosing the tools to do so.

CSC309: Programming on the Web

Winter 2022

Topics included both front-end and back-end web development. We created dynamic front-end webpages, and backends using Express servers and MongoDB. The two main projects involved developing a fully functioning web app and a front-end JavaScript library.

CSC384: Introduction to Artificial Intelligence

Winter 2022

Some topics we covered include search, constraint satisfaction, knowledge representation, game tree search and reasoning with uncertainty. For our assignments, we implemented several algorithms in Python and designed our own heuristics. One of our assignments included implmenting an AI player for Othello.
